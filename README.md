# Toast IQ's Next 100K Users Won't Type a Question

A work sample for Toast's MBA Product Manager Intern role by [Sidharth Sundaram](https://sidharthsundaram.com).

## What This Is

An interactive analysis exploring Toast IQ's next growth frontier: the 25K restaurants driving early traction are data-curious and English-fluent, but the next wave of operators — international, non-English US, smaller SMBs — will need a different interaction path than conversational AI.

Includes an interactive before/after demo of a proposed "Guided Mode" — visual insight cards with one-tap actions as a complementary path alongside the current conversational interface.
